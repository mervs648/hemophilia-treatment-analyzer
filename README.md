# hemophilia-treatment-analyzer
A Python-based decision support tool for hemophilia patient management and cost optimization# Hemophilia Patient Treatment & Cost Analyzer

This Python project is a decision-support tool designed to analyze hemophilia patient data, classify disease severity, and optimize medication dosages.

## Key Features
* **Severity Classification:** Automatically determines severe, moderate, or mild status based on factor-8/9 levels.
* **Dosage Algorithm:** Calculates optimal factor units and vial distribution (250IU - 2000IU).
* **Financial Analysis:** Estimates 4-week and annual medication costs covered by SSI.
* **Statistical Reporting:** Generates comprehensive population metrics including inhibitor presence.
